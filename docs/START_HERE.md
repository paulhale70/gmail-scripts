## ⚡ Quick Install (3 Steps)

### 1. Get the Code
- Go to [GmailAnalyzer.gs](GmailAnalyzer.gs)
- Click **Copy** button
- Code is now copied!

### 2. Create Google Sheet
- Go to [sheets.google.com](https://sheets.google.com)
- Click **+ Blank**
- Name it "Gmail Analyzer"

### 3. Add Script
- Click **Extensions** > **Apps Script**
- Delete sample code
- Paste your code (Ctrl+V)
- Click **Save**

**Need more detail?** → [BEGINNER_GUIDE.md](BEGINNER_GUIDE.md)

## ❓ Common Questions

### Is this safe?
**YES!** The code runs in YOUR Google account only. No data leaves Google. You can review all the code before using it.

### Do I need to pay?
**NO!** Everything is completely free. Google Sheets and Apps Script are free.

### Do I need a GitHub account?
**NO!** You can download the code without creating an account.

### Do I need to know coding?
**NO!** Just copy and paste. The code is already written for you.

### Will it delete my emails?
**Only if you tell it to!** All destructive actions require your confirmation.

### Can I undo actions?
**Yes!** Deleted emails go to trash (30-day recovery). You can also use "Archive" instead of "Delete" for safety.

## 🌟 Pro Tips

### For Beginners
1. **Start small** - Analyze 30 days before trying 90 days
2. **Use Archive first** - Safer than Delete while learning
3. **Read the docs** - Most questions are answered in guides
4. **Take your time** - No rush, go at your own pace

### For Everyone
1. **Review before deleting** - Always check what will be removed
2. **Export regularly** - Back up important data to CSV
3. **Check duplicates first** - Easy wins for storage space
4. **Unsubscribe ruthlessly** - Less email = less stress

### For Power Users
1. **Read EXAMPLES.md** - Lots of customization ideas
